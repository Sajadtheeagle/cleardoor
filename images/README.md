## Image Guidelines

- **Format:** WebP preferred, JPG fallback. PNG for logos/icons with transparency
- **Compression:** Target <150KB for hero images, <80KB for thumbnails
- **Blog thumbnails:** 800×450px (16:9)
- **Listing photos:** 900×600px
- **OG images:** exactly 1200×630px
- **Agent headshots:** 400×400px (square, face-centered)
